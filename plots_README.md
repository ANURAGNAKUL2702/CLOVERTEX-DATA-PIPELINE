# Plots Reference

This file describes the PNG outputs generated by `pipeline/visualization/plots.py`.

## Output Directory

`datalake/consumption/plots/`

## Plots

- `age_histogram.png`
  - Patient age distribution from the unified dataset.

- `gender_bar.png`
  - Gender distribution from the unified dataset.

- `diagnosis_frequency_chapters.png`
  - Top 15 ICD-10 chapters by patient count (horizontal bar chart).

- `lab_distribution_reference.png`
  - Lab value distributions for HbA1c and creatinine with reference range boundaries.

- `genomics_scatter_significance.png`
  - Allele frequency vs read depth colored by clinical significance.

- `high_risk_summary.png`
  - High-risk vs other patient counts (HbA1c > 7.0 and pathogenic variants).

- `data_quality_overview.png`
  - Summary of nulls handled, duplicates removed, orphan records, schema mismatches.
